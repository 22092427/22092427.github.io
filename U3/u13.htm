What is Unit 13?

Unit 13 titled Website Development is about understanding the uses and features of websites, before moving on to the designing of a website that is aimed at anyone who is interested in buying a mobile phone.

After the design process it was time to move onto the creation of the website, and once completed and built, move on to the testing phase. The testing phase helped to ensure that all the functions and display of the website was working correctly as intended. After gathering and considering feedback from 3 individuals, I made some refinements to my website so that it was more user friendly and better alligned with the user requirments, purpose and intended audience.
